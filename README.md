# 🚀 StarApp Builder

Convert your **GitHub Pages website into a real Android APK** automatically using cloud-based build (no Android Studio required).

---

## 🌐 Live Website

👉 https://starverse1130.github.io/Star-Name/

---

## 📱 What This Project Does

This project creates a **clean Android WebView app** that:

* Loads your website in full screen
* Feels like a real mobile app
* Works without any extra UI clutter
* Supports back navigation
* Can be extended with update system (version control)

---

## ⚙️ Tech Stack

* Android (WebView आधारित app)
* Java
* Gradle
* GitHub Actions (APK build automation)

---

## 🚀 How It Works

1. Website is hosted on GitHub Pages
2. Android app loads the live URL
3. GitHub Actions builds APK in cloud
4. APK can be downloaded and installed
